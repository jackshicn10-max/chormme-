# v0.1 使用检查清单

实际使用时，优先用这份清单排查。

## 安装

- 扩展已从开发者模式加载
- 加载目录是 `build\chrome-mv3-prod`
- 重新 build 后已经点过“重新加载”

## 视频页

- 当前页面是 `youtube.com/watch?...`
- YouTube 字幕菜单里已切到英文字幕或英文自动字幕
- 播放器左下区域能看到 `LT OFF / LT ON`

## 字幕

- 打开 LT 后右侧字幕面板出现
- 当前字幕轨标签显示正常
- 如果没载入，先点“刷新字幕”
- 若仍失败，先去 YouTube 字幕菜单重新切到英文，再刷新

## 查词

- hover 单词有释义
- 释义特别慢时，点字幕面板里的“预热释义”，先缓存后续 20 分钟单词
- 释义明显错误时，优先判断是不是在线兜底返回质量差

## 生词本

- 右键单词后有 toast
- 生词本里能看到 `Word / Meaning / Sentence / Context / Video`
- 导出 CSV 和 JSON 正常

## 设置

- 设置页能打开
- 默认 LT 开关按预期工作
- 字幕提前量建议从 0 开始；修改后刷新页面或重新打开 LT 再观察
- 需要重置查词时，清空翻译缓存
