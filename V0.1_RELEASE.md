# Tools for Learning English in YouTube v0.1

发布日期：2026-04-23

## 当前版本定位

这是一个已经可实际使用的 `v0.1` 浏览器扩展版本，用来替代对 Language Reactor 的核心依赖。

当前目标不是“功能最全”，而是：

- 能稳定在 YouTube 视频页拉到英文字幕
- 能在右侧显示完整字幕列表
- 能用自己的播放器字幕覆盖 YouTube 那种一个字一个字蹦出的显示
- 能右键存词，并带上下文、视频信息、时间戳
- 能 hover 查词，并有基础中文释义
- 能导出数据
- 能用设置页调整默认行为

## v0.1 已实现

- YouTube watch 页面 LT 开关
- 右侧字幕面板
- 刷新字幕按钮
- 自动识别当前英文字幕轨道
- 自动生成字幕和手动字幕都可尝试载入
- 可读化字幕合并
- 自定义播放器整句字幕覆盖
- 字幕提前量补偿
- 单词 hover 中文释义
- 内置词典 + 缓存 + 在线兜底
- 当前视频单词预热
- 从当前时间点手动预热后续释义
- 右键加入生词本
- 生词本页
- CSV / JSON 导出
- 设置页

## 设置项

- 进入 YouTube 视频页后默认打开 LT
- 启用鼠标悬停查词
- 字幕加载后预热当前视频单词
- 手动预热释义范围：后续 20 分钟 / 本视频剩余全部
- 字幕提前量
- 清空翻译缓存

## 已知问题

- 某些视频的英文自动字幕轨本身会卡住，插件只能跟着失败，需要手动切字幕或点“刷新字幕”。
- 在线兜底翻译接口质量一般，所以优先依赖内置词典和缓存。
- 默认 LT 开关是否自动打开，对已经打开的 YouTube 标签页通常需要刷新页面后完全生效。
- 改设置后，YouTube 当前页的行为一般需要刷新或重新开关 LT 才会彻底同步。

## 推荐使用方式

1. 打开 YouTube 视频页
2. 确认字幕菜单里选的是英文字幕或英文自动字幕
3. 观察播放器里的 `LT OFF / LT ON`
4. 打开 LT
5. 如果字幕没出来，点右侧“刷新字幕”
6. 悬停看中文释义，右键保存生词
7. 需要整理时打开生词本导出
